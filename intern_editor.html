עורך נתוני שיבוץ

    בחר מתמחה מהרשימה, או הוסף חדש.
    הזן את שם תורן החוץ בתא של היום והעמדה שהוא מכסה. כל תא מלא = משבצת שמוסרת משיבוץ המתמחים (מה שמפחית את העומס). הייצוא יוצר קובץ external_<חודש>.csv שמועבר לסקריפט כארגומנט החמישי.
    ערב חג (כמו שישי) חג (כמו שבת)
    לחיצה על יום מסמנת אותו: רגיל ← ערב חג ← חג ← רגיל. ערב חג מתנהג כשישי והחג כשבת לצורכי איזון סופי־שבוע ותקרות. אפשר להוסיף שם לכל חג ברשימה למטה. הייצוא יוצר holidays_<חודש>.csv.
    השיבוץ מיוצר על ידי שרת שמריץ את monthly_scheduler.py. יש לפרוס את השרת פעם אחת (ראו קובץ README שצורף), להדביק כאן את הכתובת שלו, ואז ליצור שיבוצים ישירות מהדפדפן ללא הורדה/העלאה ידנית של קבצים.

    הגדרות שרת

    הכתובת והסיסמה נשמרות בדפדפן שלך (localStorage בעמוד זה בלבד) כדי שלא תצטרך להזין אותן שוב.

    יצירת שיבוץ

    משתמש בנתוני המתמחים / תורני החוץ / החגים שערוכים כרגע בעורך (לא צריך לייצא קבצים בנפרד).
